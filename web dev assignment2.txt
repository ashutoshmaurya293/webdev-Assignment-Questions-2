Answer: vw: This unit is based on the width of the viewport and A value of 1vw is equal to 1% of the viewport width.
vh: This unit is based on the height of the viewport and A value of 1vh is equal to 1% of the viewport height.

Q.5 How is Border-box different from Content Box?
Answer: Border-box means that the width and height of an element include the content are and The padding and border are included inside the element’s dimensions.
Content-box is the default value of box-sizing and It means that the width and height of an element only include the content area, and do not include the padding and border
Q.6 What’s z-index and How does it Function ?
Answer: The z-index CSS property sets the z-order of a positioned element and its descendants or flex items and Overlapping elements with a larger z-index cover those with a smaller one.

Q.8 Difference between absolute and relative and sticky and fixed position explain with example.
Answer: Static is the default value of position.
Relative is similar to static positioning, but the element is positioned relative to its normal position
Absolute positions an element relative to its nearest positioned ancestor (instead of relative to the viewport like fixed)
Fixed positions an element relative to the viewport (the browser window), which means it always stays in the same place even if the page is scrolled
Sticky is a relatively new value for position that combines aspects of both relative and fixed positioning
